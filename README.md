## Run Locally

### Step 1: Clone project

```sh
git clone https://github.com/MikhailEzhov/kanban_board.git
cd kanban_board
```

### Step 2: Install npm dependencies

```sh
npm install
```

### Step 3: Usage

```sh
npm start
```
